Methodology

Signal model

The system scans a defined universe for securities in trend that pull back toward EMA zones without breaking longer-term structure.

Spread selection

Defined-risk debit and credit spreads are evaluated by structure fit, risk/reward, DTE, volatility, and modeled setup quality.

Liquidity model

Contracts are screened using bid/ask width, quote quality, open interest, volume, and fillability constraints.

Risk filters

The public layer excludes account sizing, broker data, order status, private scoring weights, and exact executable instructions.
